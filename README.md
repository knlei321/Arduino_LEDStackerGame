# Arduino_LEDStackerGame

這是一個基於 Arduino 平台開發的經典 **Stacker (疊疊樂)** 遊戲。玩家需要精準點擊按鈕來堆疊移動中的 LED 方塊，來讓方塊疊得更高!

## 專案簡介

這項專案使用 MAX7219 LED 矩陣（4 合 1 模組）呈現。
核心邏輯包含：
- **動態削減機制**：若方塊未對準下方堆疊，沒對齊的部分會自動縮減，增加遊戲難度。
- **自動加速**：隨著層數增高，方塊移動速度會逐漸加快。
- **除錯模式**：支援透過 Serial Monitor 進入測試模式，確保硬體接線正確。

## 硬體需求
- **Arduino Uno / Nano** (或其他相容開發板)
- **MAX7219 8x32 LED 矩陣** (FC-16 型號)
- **輕觸按鈕** x 1
- **杜邦線** 若干

## 🚀 如何開始
1. 安裝 Arduino 函式庫：`MD_MAX72xx`。
2. 開啟 `LEDgame.ino`。
3. 確保 `MAX_DEVICES` 設定為 4。
4. 上傳程式碼至 Arduino。
5. 按下按鈕開始遊戲！

## 🕹️ 控制說明
- **按下按鈕**：鎖定目前方塊並進入下一層。
- **遊戲結束**：畫面閃爍，按住按鈕再放開即可重新開始。
